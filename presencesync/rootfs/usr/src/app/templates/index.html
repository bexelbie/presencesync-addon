Two-factor code

    
A 6-digit code should have appeared as a notification (Apple ID Sign-In Requested) on your trusted Apple devices. Tap Allow to reveal it.

    

      Notification didn't arrive?
      

        Apple sometimes silently suppresses push notifications for third-party auth. To get a code manually:
      

      
    
        
  1. Put the iPhone/iPad you're getting the code from into Airplane Mode
    2. 
        
  2. Open Settings → tap your name at the top → Sign-In & Security → Get Verification Code
    3. 
        
  3. A 6-digit code displays — enter it below, then submit before exiting Airplane Mode
    4. 
      

      
The Airplane Mode step prevents iOS from auto-completing the flow while you're typing the code into PresenceSync.
