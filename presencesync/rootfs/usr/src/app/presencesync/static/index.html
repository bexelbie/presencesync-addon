Apple Account

Enter the 6-digit verification code.
No push notification? That's normal — unofficial API clients often don't trigger one. Generate a code manually:
iPhone: Settings → [your name] → Sign-In & Security → Get Verification Code
Tip: If the code keeps failing, try putting your phone in Airplane Mode briefly, generate the code, then turn Airplane Mode off and submit it. This prevents Apple from auto-consuming the code via push before you can enter it.

Key Extraction

Extract AirTag private keys from iCloud Keychain. This is a separate authentication from the login above — Apple requires it because the extractor joins the Keychain trust circle directly.

Step 1 of 5 — Authenticate to iCloud Keychain

Step 2 of 5 — A new 2FA code is needed because the extractor authenticates as a separate device. Check your trusted device for a fresh code.
Tip: Put your phone in Airplane Mode, generate the code (Settings → [your name] → Sign-In & Security → Get Verification Code), then turn Airplane Mode off and submit.

Step 3 of 5 — Choose which device's escrow backup to use. Pick a device whose passcode you know.

Step 4 of 5 — Enter the screen lock passcode of the device you selected above.
